# RA6M5_Smart-check-in
> 26全国大学生电子设计竞赛信息科技前沿专题赛入围国赛瑞赛项目作品 - 多端感知智能打卡系统

---

## 📖 项目简介
本项目为全国大学生电子设计竞赛作品，基于瑞萨 RA6M5 微控制器开发的多端感知智能打卡系统。
包含主控代码、ESP32网络模块、HMI串口屏交互、K230模型识别四大模块，实现了一套完整的智能打卡解决方案。

## 🛠️ 开发环境
- **主控平台**：瑞萨 RA6M5
- **开发工具**：e²studio / Renesas RA Smart Configurator
- **通信方式**：UART / WiFi
- **外设支持**：HMI串口屏、ESP32、K230视觉模块
